# 📈 B2B Retail Demand Optimization Dashboard
An enterprise intelligent replenishment and order volume recommendation engine built using an optimized Random Forest Regressor and a modular scikit-learn preprocessing pipeline.

## 🛠️ Local Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Generate data: `python src/generate_data.py`
4. Preprocess data: `python src/preprocessing.py`
5. Train model: `python src/train.py`
6. Run interface: `python app.py`
